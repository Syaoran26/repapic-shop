## Chạy project

### 1. Cài NodeJS

```sh
npm install npm@latest -g
```

### 2. Cài Node modules

```sh
  npm install
```

### 3. Khởi chạy

- Front-end

  ```sh
    npm start
  ```

- Back-end
  ```sh
      npm run dev
  ```
